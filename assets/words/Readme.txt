===== Read me =====

Each line is 1 word. for default first part is english vocabulary
and second is meaning of it. the vocabulary mustn't has a blackspace.
if it has, the second part of it will become meaning and the true
meaning in the third part will be useless.

File only work with name 'engmix_lv1.txt' - 'engmix_lv5.txt'

Don't forget to check the last line of the file too. the empty line will
make a game error and crash

What you can do with these files?
- add more vocabulary. by add more line.
- edit all vocabulary and make new arrange.
- swap word and meaning in file. so the game will show english word
  and you need to input meaning as answer.
- etc.

===== โปรด อ่าน =====

แต่ละบรรทัดของไฟล์คือคำหนึ่งคำ โดยเริ่มต้น ส่วนแรกของบรรทัดจะเป็นคำศัพท์ภาษาอังกฤษ และส่วนที่สองจะเป็น
ความหมายของคำนั้น ถ้าหากจะแก้คำหรือเพิ่มคำ คำที่เพิ่มเข้ามาจะต้องไม่มีช่องว่าง หรือการเว้นวรรคทั้งสองส่วน
ไม่เช่นนั้น ส่วนที่สองที่แยกมาของคำศัพท์จะกลายเป็นความหมายที่แสดงในเกม และส่วนที่สามซึ่งเป็นความหมายจริงๆนั้น
จะไม่ได้ถูกนำไปใช้

ไฟล์ที่จะถูกนำไปใช้มีแค่ไฟล์ที่ชื่อ 'engmix_lv1.txt' ถึง 'engmix_lv5.txt' เท่านั้น

และโปรดระวังเรื่องบรรทัดเปล่าในบรรทัดสุดท้ายด้วย ถ้าหากมีบรรทัดเปล่าในบรรทัดสุดท้าย เกมจะเกิดปัญหาขึ้นและ
ทำงานไม่ได้

แล้วทำอะไรกับไฟล์คำศัพท์ได้บ้าง?
- สามารถเพิ่มคำศัพท์เข้าไปใหม่ได้ โดยการเพิ่มในบรรทัดใหม่
- แก้ไขคำทั้งหมดแล้วจัดเรียงเป็นหมวดหมู่ใหม่
- สลับตำแหน่งของคำภาษาอังกฤษและความหมาย ถ้าสลับเกมจะแสดงส่วนที่เป็นคำภาษาอังกฤษ และเราต้องพิมพ์ตอบ
  คำตอบที่เป็นความหมายลงไปแทน
- เป็นต้น
